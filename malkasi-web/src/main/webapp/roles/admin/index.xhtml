En el sistema de administración usted puede hacer gestión de usuarios, ver las publicaciones y proyectos de invetigación del DIINF, realizar gráficos y obtener estadísticas.





Este sistema permite generar estadísticas y tablas de resumen acerca de las actividades que se esten realizando en el departamento, ya sean, publicaciones o proyectos de investigación.





Log permite saber que acciones han realizado los usuarios del sistema en el contexto de las publicaciones
